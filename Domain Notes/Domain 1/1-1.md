# 1.1 Notes

## Controls

- **Technical**
  1. Firewalls — protect computer networks by filtering traffic based on rules.
  2. Data encryption — converts sensitive data into unreadable data without the correct key.

- **Managerial**
  1. Performance reviews — assessments of employee performance.
  2. Risk assessments — identification, evaluation, and mitigation of risk.
  3. Code of conduct — a set of guidelines and ethical standards.

- **Operational**
  1. Incident response procedures — controls that outline the steps to be followed if a security event happens.
  2. Security awareness training — education about security threats.
  3. User access management — management of user access.

- **Physical**
  1. Access control — door access.
  2. Biometric locks — use physical characteristics (fingerprint, iris, etc.) to authenticate.
  3. Guard personnel — a visible deterrent and physical involvement.
  4. Security fencing — physical barriers.
  5. CCTV surveillance — monitors and records activity.
  6. Mantraps — an enclosed area allowing only one person in and one person out at a time.
  7. Vehicle barriers — prevent vehicles from entering.
  8. Tamper seals — used to secure containers.
  9. Panic buttons — alarm to alert authorities.

- **Control types**
  1. Preventive controls — prevent a risk from happening.
  2. Deterrent controls — discourage someone from engaging in an action.
  3. Detective controls — uncover risks that have already happened.
  4. Corrective controls — fix a problem after an event has happened.
  5. Compensating controls — an alternative measure used when the primary control is not possible.
  6. Directive controls — policies or guidelines used to ensure compliance.

## Some questions to answer

1. What is the difference between a preventive control and a detective control? Give one example of each.
2. Which category of control (technical, managerial, operational, or physical) does a firewall belong to, and why?
3. A company can't patch a legacy system right away — what type of control would they use in the meantime, and what does that control type mean?
4. Name two physical controls that would stop an unauthorized person from following an employee through a secured door, and explain how each works.
5. What's the difference between a corrective control and a compensating control?
6. Which control type is a security awareness training program, and which control category does it fall under?
7. Give an example of a directive control and explain why it relies on policy rather than technology to work.
8. If a break-in already occurred and cameras recorded the event, which control type captured that evidence — and what control would have tried to stop the break-in in the first place?
